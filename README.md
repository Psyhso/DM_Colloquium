## Цель:
Реализовать систему компьютерный алгебры, со следующими модулями:
  1. Натуральные числа.
  2. Целые числа.
  3. Рациональные числа.
  4. Полиномы.
Конечный продукт - сайт, для взаимодействия с созданными модулями.

## Инструменты:
- Язык программирования: Python. Выбран Python, так как все на достаточном уровне владеют навыками программирования на данном языке. Версия Python 3.11.9.
- Фреймворк для создания веб-приложения - Flask. Выбран в силу своей простоты и возможности быстро и качество реализовать backend состоявляющую.
- Тестировние реализовано при помощи библиотеки Pytest. Выбрана в силу того, что тесты пишутся быстро и не нужно углубляться в саму библиотеку.

## Ход выполения работы:
- Для начала были реализованы все методы, не зависящие от других, заложена база. Для работы с каждым модулем был создан класс, который внутри себя инициализирует объект (число/полином). Каждый класс, который внутри себя содержит другие классы, пользуется объектами класса.
- Для написания методов, что зависят от других, разработка велась следующим образом: каждому человеку дается определенное количество методов, что идут подряд, чтобы он не зависел от других разработчиков и строгий срок выполнения задачи. Таким образом были избежены накладки и разработчики не зависили от других.
- После написания всех методов началась разработка веб-приложения. Оно из себя представляет калькулятор, в который вводятся математические выражения, в зависимости от выбранного модуля для вычислений.
- Все методы, написанные разработчиками покрывались тестами в момент написания самого метода.

## Как запустить веб-приложение? (Гайд для Windows)

- Для начала нужно клонировать репозиторий:
```
git clone https://github.com/Psyhso/DM_Colloquium.git
```

- После того как склонировали репозиторий создайте вирутальное окружение и установите зависимости. Не забудьте про версию Python 3.11.9 (на остальных версиях возможна некорректная работа программы).
  Чтобы создать виртуальное окружение, перейдите в папку с репозиторием
```
python -m venv venv
```

- Теперь необходимо активировать виртуальное окружение и установить зависимости:
```
venv\Scripts\activate
pip install -r requirements.txt
```

- Теперь все готово к запуску, есть два варианта:
  1. Открыть через IDE файл app.py и запустить
  2. Запустить программу через консоль
```
python src\app\app.py
```
